warp.\_src.lang.texture\_sample
===============================

.. function:: warp._src.lang.texture_sample(tex: Texture1D, u: float32, dtype: Any) -> Any

   .. hlist::
      :columns: 8

      * Kernel

   Sample the 1D texture at the given U coordinate.
   
   .. admonition:: Experimental
   
       The texture API is experimental and subject to change. See :class:`warp.Texture`.
   
   :param tex: The 1D texture to sample.
   :param u: U coordinate. Range is [0, 1] if the texture was created with
             ``normalized_coords=True`` (default), or [0, width] if ``normalized_coords=False``.
   :param dtype: The return type (``float``, :class:`warp.vec2f`, or :class:`warp.vec4f`).
   
   :returns: The sampled value of the specified ``dtype``.
   
   Filtering mode is :attr:`warp.TextureFilterMode.CLOSEST` or :attr:`warp.TextureFilterMode.LINEAR`.
