warp.\_src.lang.norm\_pseudo\_huber
===================================

.. function:: warp._src.lang.norm_pseudo_huber(v: Any, delta: float) -> float

   .. hlist::
      :columns: 8

      * Kernel
      * Differentiable

   Compute the "pseudo" Huber norm of a vector v with a given delta.
   
   .. math::
       H^\prime(v) = \delta \sqrt{1 + \frac{\|v\|^2}{\delta^2}}
   
   .. image:: /img/norm_pseudo_huber.svg
       :align: center
   
   :param v: The vector to compute the Huber norm of.
   :type v: Vector[Float, Any]
   :param delta: The threshold value, defaults to 1.0.
   :type delta: float
   
   :returns: The Huber norm of the vector.
   :rtype: float
