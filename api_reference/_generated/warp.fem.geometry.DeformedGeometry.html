Geometry obtained by deforming a base geometry with a displacement field.

__init__(field, relative=True, build_bvh=False)[source]#

Construct a Deformed Geometry from a displacement or absolute position field defined over a base geometry. The deformation field does not need to be isoparameteric.

See also: warp.fem.GeometryField.make_deformed_geometry()
