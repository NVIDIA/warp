warp.tile\_view
===============

.. function:: warp._src.lang.tile_view(t: Tile[Any, tuple[int, ...]], offset: tuple, shape: tuple[int, ...]) -> Tile[Any, tuple[int, ...]]

   .. hlist::
      :columns: 8

      * Kernel

   Extract a view of a tile.
   
   ``offset`` may contain integer coordinates, in which case ``shape`` gives the
   returned tile shape. ``offset`` may also contain ``slice`` objects, in which
   case the returned shape is inferred from the slice bounds and ``shape`` must
   not be specified.
   
   :param t: Input tile to extract a subrange from
   :param offset: Integer offsets or slices in the source tile
   :param shape: Shape of the returned view for integer-only offsets
   
   :returns: A tile view with dimensions given by ``shape``, the remaining source tile
             dimensions, or the inferred slice extents.
